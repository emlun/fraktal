fraktal
===

Fractal rendering sandbox, just for fun.

Try it here: https://emlun.se/fraktal/


Usage
---

```
$ wasm-pack build
$ cd www
$ npm install
$ npm start
$ $BROWSER http://localhost:8080
```


Controls
---

The following is subject to change.

- Mouse drag: Pan view
- Mouse wheel up: Zoom in
- Mouse wheel down: Zoom out
